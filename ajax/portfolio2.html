maxVal
Full Stack MERN Application
Description

A premier crowdsourced product valuation application for modern market research. maXval enables individuals and companies of any size to quickly evaluate their products in a simple and approachable platform.

Users are able to upload photos of their products, which are then rated based on other user's input. The ratings are stored in the database and are used to populate the product dashboard.

This application was designed using React.js on the front-end and MongoDB, node.js, and express.js on the back-end. The user authentication was built using Firebase, and charts.js was utilized to create the data visualization.
