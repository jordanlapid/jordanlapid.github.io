Clicky Game!
A Memorization Game in React
Description

Clicky Game is a game designed to test your memory! The object of the game is to click all the pictures without clicking the same one twice.

The app was built using React for the UI, javascript for the logic and was deployed on Heroku.

Details
  • TechnologiesReact.js, Javascript, HTML, CSS, Heroku
