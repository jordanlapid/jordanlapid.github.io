Comic Book Gifs!
Bootstrap/Giphy API Web Application
Description

Comic Book Gifs! is an application that generates GIFs based on user input. AJAX calls are made to the Giphy API when buttons are pressed, and the GIFs are dynamically generated using jQuery.
