Nick Trivia
Mobile Responsive Trivia Game
Description

Nick Trivia is a game designed to test your "Golden-Age" Nickelodeon knowledge. The player has 15 seconds to answer each question, and the correct answer is displayed after a choice is made. Score is kept throughout the game is displayed at the end!
