Bucket List
Full Stack Web Application
Description

An app designed to allow users to create and save their bucket lists using a photo.

When a user uploads their photo, Google Cloud Vision API analyzes the photo and returns information, which is used to populate the bucket list item.

This app uses passport.js for user authentication, mySQL and node.js for the back-end, and jQuery and handlebars to render the front-end content.
