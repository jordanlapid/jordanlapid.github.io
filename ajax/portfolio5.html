Bamazon
Node.js/MySQL CLI Application
Description

Bamazon is an app that uses the command line interface to track inventory and allows a user to purchase items, which decreases the inventory and gives a total prices to the user.

This app was built using node.js and uses inquirer module to prompt the questions. mySQL is used to store and update the inventory data based on user input.
